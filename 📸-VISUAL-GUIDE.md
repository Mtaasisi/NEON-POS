## 🎨 Logo Tips

### ✅ Good Logo Examples:

```
Option 1: Square Logo (Best)
┌────────────┐
│    LATS    │
│   CHANCE   │
│    STORE   │
└────────────┘
200x200px, transparent background

Option 2: Horizontal Logo (Good)
┌──────────────────────────┐
│  LATS CHANCE STORE       │
└──────────────────────────┘
400x100px, transparent background

Option 3: Logo with Icon (Great)
┌────────────┐
│    🏪      │
│   LATS     │
└────────────┘
200x200px, transparent background
```

### ❌ Avoid:

```
Too Large:
┌────────────────────────────────┐
│         HUGE LOGO              │
│    (over 2MB)                  │
│         ❌                      │
└────────────────────────────────┘

Too Detailed:
┌────────────────────────────────┐
│ [Complex image with tiny text] │
│ Won't print well on receipts   │
│         ❌                      │
└────────────────────────────────┘

Wrong Format:
• .docx files  ❌
• .pdf files   ❌
• .svg files   ❌ (use PNG instead)
```

## 🔍 Quick Checklist

```
Before You Start:
☐ Have your logo file ready
☐ Logo is JPG, PNG, GIF, or WebP
☐ File size is under 2MB
☐ Logo dimensions are reasonable (200x200px recommended)

After Database Migration:
☐ See "INSTALLATION COMPLETE!" message
☐ All fields show ✅ checkmarks
☐ No error messages

After Uploading Logo:
☐ Logo preview appears immediately
☐ Clicked "Save Settings" button
☐ See success toast/message
☐ Generated test receipt
☐ Logo appears on receipt

If Something Goes Wrong:
☐ Check browser console (F12) for errors
☐ Re-run the SQL migration
☐ Refresh browser (Ctrl+Shift+R)
☐ Try a different logo file
☐ Check file size and format
```
