================================================================================
                          TROUBLESHOOTING
================================================================================

Problem: Gradle sync fails
---------------------------
Solution:
1. File > Invalidate Caches > Invalidate and Restart
2. Tools > SDK Manager
3. Install: Android SDK Build-Tools (latest)
4. Try again

Problem: Emulator won't start
------------------------------
Solution:
1. Try different API level (API 30 or 31)
2. Tools > AVD Manager > Delete old emulator
3. Create new one
4. Make sure virtualization enabled in BIOS

Problem: App crashes on launch
-------------------------------
Solution:
1. Check Logcat for error messages
2. Look for red lines
3. Common issues:
   â†’ Missing permissions
   â†’ Network configuration
   â†’ Database connection

Problem: Black screen
---------------------
Solution:
1. Wait 10-15 seconds (might be loading)
2. Check chrome://inspect for JavaScript errors
3. Rebuild: Build > Clean Project > Rebuild Project

Problem: "App not installed"
-----------------------------
Solution:
1. Uninstall any existing version
2. Build > Clean Project
3. Run again

================================================================================
                          PERFORMANCE TIPS
================================================================================

For faster emulator:
- Use x86/x86_64 system images (faster than ARM)
- Allocate more RAM (Settings > Advanced)
- Enable hardware acceleration
- Use Quick Boot (enabled by default)

For faster builds:
- Build > Make Project (instead of Run)
- Use instant run if available
- Close unnecessary apps
