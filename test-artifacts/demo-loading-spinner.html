üìä Product Card Information Guide

üé® Badge Colors

Purple: New condition
Green: Excellent / Unlocked
Blue: Good condition / Color
Yellow: Storage capacity
Orange: Trade-in source
Indigo: RAM / Tech specs
Red: Low stock alert

üì¶ Product Details

IMEI: Unique device identifier
SKU: Stock keeping unit code
Category: Product classification
Supplier: Source/vendor info
Location: Storage position
Date Added: Inventory date
Warranty: Coverage period

üí∞ Financial Metrics

Selling Price: Customer price
Cost Price: Acquisition cost
Profit Margin: % difference
Trade-in Value: Original offer
Stock Count: Available units
Min. Stock: Reorder threshold
