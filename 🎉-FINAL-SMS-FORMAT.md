## 🎨 Emoji Meanings

- 🧾 = Receipt document
- 📅 = Date and time
- 📦 = Items/products
- 💰 = Total amount
- ✨ = Thank you sparkle

**Makes it friendly and easy to scan!**

## 📊 Format Breakdown

### Header:
```
🧾 RECEIPT #S-001
📅 10/11/2025, 2:30 PM
```

### Items Section:
```
📦 Items: 3
1. Product Name - Price TZS
2. Product Name x2 - Price TZS
3. Product Name - Price TZS
```

### Footer:
```
💰 TOTAL: Amount TZS

✨ Thank you for shopping with us!
```
