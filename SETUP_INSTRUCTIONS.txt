STEP 1: RUN DATABASE MIGRATION
===============================================

Apply the comprehensive database migration:

Or if using Supabase, run the SQL in the Supabase SQL editor.

This will create:

- Communications table

- Categories & tags tables
- Contracts table
- Plus all necessary indexes, triggers, and views

STEP 2: SET UP STORAGE BUCKET (For Documents)
===============================================

In Supabase Dashboard:
1. Go to Storage
2. Create a new bucket named: supplier-documents
3. Set it to PUBLIC or configure RLS policies
4. Allow file uploads for authenticated users

===============================================
USAGE EXAMPLES
===============================================

1. VIEW SUPPLIER DETAILS:
   - Click on any supplier card
   - Navigate through tabs (Overview, Documents, etc.)
   - All information in one place

2. UPLOAD DOCUMENTS:
   - Go to Documents tab
   - Click "Upload Document"
   - Select file, type, and expiry date
   - Track all supplier documents

3. LOG COMMUNICATION:
   - Go to Communications tab
   - Click "Log Communication"
   - Record interaction details
   - Set follow-up reminders

4. RATE SUPPLIER:
   - Go to Ratings tab
   - Click "Add Rating"
   - Rate overall + specific categories
   - Add review text and pros/cons

5. MANAGE CONTRACTS:
   - Go to Contracts tab
   - Click "New Contract"
   - Set dates, value, and terms
   - Get expiry notifications

6. TRACK FINANCES:
   - Go to Financial tab
   - Set credit limits
   - Monitor balances
   - Configure payment terms

7. IMPORT SUPPLIERS:
   - Click "Import" button
   - Download template CSV
   - Fill in supplier data
   - Upload CSV to import

8. EXPORT DATA:
   - Click "Export" to download all suppliers
   - Click "Stats" for statistics report
   - Data in CSV format
